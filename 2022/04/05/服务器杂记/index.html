-bash: /bin/mv: Argument list too long的解决方案

在linux服务器移动文件的时候，我们一般是 mv 源文件 目标目录，如

[…..]# mv ../blog/* .

-bash: /bin/mv: Argument list too long

上面mv的时候会报参数过长，解决方案如下：

find source file_path -type f -name ‘*. jpg’ -exec mv {} target_path ;
